Why HTML is needed

“HyperText Markup Language” is the language used to tell your web browser what each part of a website is. So, using HTML, you can define headers, paragraphs, links, images, and more,
so your browser knows how to structure the web page you're looking at.

There are some good free HTML editor

  • Atom
  • Sublime
  • Brackets
  • GNU Emacs
  • Visual Studio Code
  • BBEdit
  • Komodo Edit
  • NetBeans
Shortcut keys for ATOM
  • ctrl+z = cancel recent order
  • ctrl+s = save
